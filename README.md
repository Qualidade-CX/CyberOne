# CyberOne ERP/SaaS Skeleton - versão revisada

Projeto React + Vite com navegação funcional e dados fake para demonstração executiva.

## Rodar

```bash
npm install
npm run dev
```

## Build testado

```bash
npm run build
```

## Correções desta versão

- Sidebar revisada: todos os módulos e submenus respondem ao clique.
- Submenus permanecem acessíveis, com estado visual ativo.
- Módulos abaixo de SAI/Agentes IA foram conectados a telas demonstrativas reais.
- Ações genéricas, exportações e cards agora dão retorno visual via toast.
- Books revisado para não cortar o painel/slide em notebook e desktop.
- Drawer operacional foi ampliado e recebeu estrutura mais parecida com sistema real.
- Cards de Book agora são clicáveis.
- Ajustes de responsividade, overflow, scrollbar, largura da sidebar e área principal.

## Observação

Ainda é um esqueleto frontend com dados fake. A próxima etapa de implantação é separar frontend/backend, criar autenticação, banco, APIs e persistência.
